# Nmap-Scans

This project demonstrates four Nmap scan types against a target Kali Linux. The scans performed are Stealth, TCP Connect, UDP and ACK. This report documents the results of each scan, explaining their methodology and compares their differences.

## Ports used
-ssh
-apache

## Types of Nmap scans
- Stealth Scan
- TCP Connect
- UPD Scan
- ACK Scan
